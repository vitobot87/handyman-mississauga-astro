# Handyman Mississauga

Professional handyman services website built with Astro and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when pushing to the `main` branch.

**Live site:** https://vitobot87.github.io/handyman-mississauga-astro/
